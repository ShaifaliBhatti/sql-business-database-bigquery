
# SQL Business Database Analysis using Google BigQuery

## 📌 Project Overview
This project demonstrates SQL database creation and business analysis using Google BigQuery.

## 🗄️ Database Tables
- Customers
- Products
- Orders

## 🛠 Tools Used
- Google BigQuery        
- SQL

## 📊 Business Questions Solved
- Total Revenue
- Top Customers by Revenue
- Revenue by Category
- Monthly Revenue Trend
- Best-Selling Products

## 🔎 Key Insights
- Electronics category generated highest revenue.
- Amit Sharma is top customer.
- Revenue increased in March compared to January.
